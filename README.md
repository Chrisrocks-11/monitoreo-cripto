# 🚀 CryptoPulse Pro - Jarvis Terminal v1.0

**CryptoPulse Pro** es una terminal de monitoreo financiero desarrollada en **Angular**, diseñada para visualizar el estado del mercado de criptomonedas en tiempo real. Este proyecto combina una estética futurista "Cyberpunk" con datos precisos provenientes de la API de CoinGecko.

## 🛠️ Características Principales

* **Sincronización en Tiempo Real:** Actualización automática de precios cada 45 segundos utilizando `HttpClient`.
* **Interfaz Adaptativa (Dark/Light Mode):** Sistema de temas con persistencia visual y efectos de sonido (SFX).
* **Visualización de Datos:** Gráficos de tendencia (Sparklines) generados dinámicamente con SVG.
* **Buscador Inteligente:** Filtro reactivo para localizar activos por nombre o símbolo.
* **Diseño Premium:** Estética basada en la fuente *Orbitron*, con efectos de lluvia de logos y feedback visual (Flash) ante cambios de precio.
* **Responsive Design:** Optimizado para escritorio, tablets y dispositivos móviles.

---

## 📦 Tecnologías Utilizadas

* **Framework:** Angular (Standalone Components)
* **Lenguaje:** TypeScript
* **Estilos:** CSS3 (Variables dinámicas y Flexbox/Grid)
* **API:** [CoinGecko API](https://www.coingecko.com/en/api)
* **Fuentes:** Google Fonts (Orbitron & Inter)

## 🔧 Instalación y Configuración

1.  Clona el repositorio:
    ```bash
    git clone [https://github.com/tu-usuario/monitoreo-cripto.git](https://github.com/tu-usuario/monitoreo-cripto.git)
    ```
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Inicia el servidor de desarrollo:
    ```bash
    ng serve
    ```
4.  Abre tu navegador en `http://localhost:4200`
